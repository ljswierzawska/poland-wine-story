# Poland Has Become Europe's Unlikely Vinophile

A data-driven story exploring how climate change and historical shifts have transformed Poland into an emerging wine-producing country in Europe.

By Luiza Swierzawska

## About

This project examines Poland's growing wine industry, from its medieval roots through communist-era suppression to the current boom of over 700 registered wineries. It includes interactive data visualizations showing production trends and regional distribution of vineyards across Polish voivodeships.
